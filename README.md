#Разработка системы детекции и распознавания текста на основе YOLOv8 и EasyOCR# 1. Описание проекта

В рамках преддипломной практики разработана система автоматической детекции и распознавания текстовых полей на изображениях документов.

Система состоит из двух этапов:

1. Детекция текстовых областей с использованием YOLOv8  
2. Распознавание текста внутри найденных областей с помощью EasyOCR  


YOLOv8, выделение bounding box, вырезание области, EasyOCR, сохранение результата в JSON.

Используемые технологии

- Python
- YOLOv8 (Ultralytics)
- EasyOCR
- PyTorch
- OpenCV
- Kaggle Notebook
- GPU (NVIDIA Tesla T4)


#Подготовка датасета

Датасет размечался в формате YOLO:

- изображения (.jpg, .png)
- текстовые файлы разметки (.txt)
- файл конфигурации data.yaml

Пример структуры датасета(можно увидеть также в прикрепленной папке yolo_export):

```
dataset/
│
├── images/
│   ├── train/
│   ├── val/
│   └── test/
│
├── labels/
│   ├── train/
│   ├── val/
│   └── test/
│
│
└──data.yaml
```
Разметка осуществлялась с помощью инструмента Label Studio. 

#Работа в Kaggle

Разработка осуществлялась в Kaggle, в котором можно бесплатно использовать GPU (30 часов в неделю).
Для того чтобы работал GPU в найтройках профиля (когда зарегистрируетесь) нужно подтвердить свои данные, а именно номер телефона. После доступен бесплатный GPU.

#Создание датасета

1. Перейти в раздел Kaggle → Datasets
2. Нажать "Create Dataset"
3. Загрузить папку с изображениями и разметкой
4. Опубликовать датасет

Загружать датасет нужно архивом zip.

#Подключение датасета в Notebook

После добавления датасета в проект, он становится доступен по пути:

/kaggle/input/название-датасета/

Для копирования датасета в рабочую директорию используется команда:

!cp -r /kaggle/input/название_датасета /kaggle/working/название папки


Это необходимо, поскольку директория `/kaggle/input` доступна только для чтения.

также можно копировать полный путь датасета к нужно папке и загрузить в kaggle output в любую папку(пример project)

!cp -r /kaggle/input/datasets/имя_пользователя/название_датасета/папка с данными /kaggle/working/project/

#Включение GPU в Kaggle

Для ускорения обучения необходимо включить графический процессор:

1. Открыть Notebook
2. Перейти в Settings
3. В разделе Accelerator выбрать GPU
4. Сохранить настройки

Проверка доступности GPU:

import torch
torch.cuda.is_available()

Если вывод: True — GPU активирован.

#Детекция и OCR

Детекция (YOLOv8)

Загрузка модели:
from ultralytics import YOLO
model = YOLO("weights/best.pt")

Предсказание:
results = model.predict(source=image, conf=0.25)

Модель возвращает bounding boxes, классы и confidence.

Распознавание текста (EasyOCR)

Инициализация:
import easyocr
reader = easyocr.Reader(['en', 'ru'], gpu=True)

Распознавание:
ocr_result = reader.readtext(cropped_image, detail=0)

#Сохранение результатов

Результаты сохраняются в JSON формате:

[
    {
        "filename": "image1.jpg",
        "detections": [
            {
                "class": "name",
                "confidence": 0.95,
                "text": "Ivan Petrov"
            }
        ]
    }
]

Для более подробного кода, советую просмотреть код.

Добавлены такие файлы:
yolo_export - основная папка, где можно посмотреть структуру датасета, который нужно добавить для самастоятельного обучения модели.
Обучение производилось на yolov8n, также можно обучить модель на таких версиях: yolov8s, yolov8m и сравнить результаты.
(для начала нужно заархивировать папку и добавить в dataset kaggle)
practica-diplom.ipynb - программный код с обучением и тестированием модели. добавляется папка yolo_export для обучения модели, после чего можно провести тестирование и получить результаты.
yolo_practica.zip -  архив, с результатами тестирования. Здесь находится результат обучения модели Yolov8, вывод результата OCR по распознаванию текста и сам файл JSON.
modelYOLOv8.zip - архив с обученной моделью, в котором находятся файлы: веса модели, папка test для добавления новых скринов документов.
testyolov8.ipynb - программный код для тестирования модели. загружаете датасет modelYOLOv8 в kaggle. запускается программный код и получаете результат распознавания текста с сохранением в файл JSON.
отчет практики.docx - документ, в котором описан отчет по проделанной работе, также с результатами и основными выводами работы.
